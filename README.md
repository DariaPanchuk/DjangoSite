# Проєкт складається з 6 основних модулів:

## 1. Polls — результат 8 туторіалів із офіційного  сайту https://docs.djangoproject.com/.

## 2. ImageRecognizer — розпізнавач зображення. 
Використовує Google Cloud Vision API.  
На вході отримує зображення (JPG, PNG, WEBP). Відправляє його у хмару Google.  
Повертає назви об'єктів (Label Detection) та рівень впевненості (Confidence Score).  
Перекладає результат за допомогою GoogleTranslator.  

## 3. VideoRecognizer — розпізнавання елементів у відео. 
Використовує згорткову нейромережу EfficientNetV2.  
На вході отримує файл з розширенням MP4 або посилання на відео в YouTube.  
Завантажує відео у тимчасову папку. Розбиває його на кадри з частотою 1 кадр/сек.  
Класифікує об'єкти на кожному кадрі. Рахує, з якою частотою зустрічається той чи інший об'єкт.  
Виводить результати, перекладені за допомогою GoogleTranslator.  

## 4. SoundRecognizer — розпізнавання звуку. 
Використовує мережу глибокого навчання YAMNet (від Google Research) на базі TensorFlow Hub.  
На вході отримує аудіофайли (WAV, MP3). Змінює частоту аудіо до 16kHz (вимога моделі).  
Нормалізація амплітуди сигналу. Видає топ-10 ймовірних звуків з відсотковою вірогідністю.  
Перекладає результат за допомогою GoogleTranslator.  

## 5. SpectrumRecognizer — цифрова обробка сигналів. Аналізує частотні характеристики звуку.
Використовує швидке перетворення Фур'є (FFT) на базі NumPy.  
На вході отримує аудіофайли (WAV, MP3). Визначає частотний спектр.  
Виділяє топ-20 домінуючих частот (в Гц). Розраховує енергетичну частку кожної частоти у загальному сигналі.  

## 6. CodeRecognizer — аналіз семантики програмного коду. 
Використовує мережу глибокого навчання codet5-base-multi-sum.  
На вході отримує файл з кодом (py, js, cpp).  
Розбиває його на логічні блоки по 1000 символів, аналізує кожен блок окремо.  
Синтезує загальний опис логіки роботи програми. Виводить його українською за допомогою GoogleTranslator.  

### Стек технологій:
* Language: Python 3.12
* Framework: Django 6.0.1
* Machine Learning: TensorFlow, PyTorch (Transformers), Keras
* Data Processing: NumPy, Pandas, Librosa, OpenCV, yt_dlp
* External APIs: Google Cloud Vision, Deep Translator (Google Translate)
* Testing: Unittest, Mock, Django Test Client
* Code helper: Gemini, ChatGPT
